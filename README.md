## **为什么搬瓦工的库存这么让人头疼**

用过搬瓦工的人都知道，这家美西老牌 VPS 商有个特别拧巴的习惯——好货永远缺货，便宜货永远秒没。

普通 KVM 套餐（DC2/DC4/DC8 那种走美国常规线路的）一般长年有货，年付 $49.99 起的入门款基本随时能买。但真正让人排队抢的，是 CN2 GIA-E、香港 CN2 GIA、日本 CN2 GIA 这类三网优化线路的方案，还有各种 PowerBox、SakuraBox、BiggerBox Pro、MegaBox Pro 之类的限量版套餐。后者经常是上午补货下午就没了，等你下班回家打开电脑一看，又是灰的。

所以"搬瓦工实时库存"这个词在圈内被搜得特别多。大家想知道的无非三件事：现在到底有没有货、什么时候补货、怎么能第一时间抢到。下面我把目前能用的几个办法都给你捋一遍。

## **搬瓦工实时库存查询的三种实用方法**

### 方法一：盯第三方实时库存监控页面（最推荐）

这是目前最省心的方式。搬瓦工官网本身没有统一的"库存总览"页面，你得一个个方案点进去才知道有没有货，特别费劲。所以圈内有几个老牌站点做了第三方汇总监控，每 5 分钟左右刷新一次，所有方案的有货/缺货状态一目了然。

目前比较活跃的两个监控页：

- **stock.bwg.net**：搬瓦工中文网维护的库存监控页面，页面清爽，按套餐分组排列，有货显示绿色"立即购买"按钮，缺货显示"补货通知"。这是我个人最常用的一个。
- **status.bwgyhw.cn**：搬瓦工优惠网维护的另一个监控页，更新频率类似，多一个补货通知订阅入口，配置信息列得更细一些，包括每个套餐的 CPU、内存、硬盘、流量、带宽一目了然。

收藏一个就够了，建议 Ctrl+D 存起来，每次想买之前先看一眼。

### 方法二：官网方案总览页手动查

如果你不想依赖第三方，搬瓦工官网的 cart 页面其实也能看库存。打开后往下拉，会看到一长串套餐列表，每个方案右侧的按钮如果是可点击的"Order Now"就是有货，灰掉的"Out of Stock"就是缺货。

缺点是页面特别长，找特定套餐要翻半天，而且限量版套餐有时候会单独放在另一个分组里，容易漏看。所以这一招适合临时确认，不适合长期蹲。

### 方法三：浏览器扩展 + 价格历史工具

部分技术型用户会用一些自动比价/库存监控的浏览器扩展，或者自己写脚本轮询搬瓦工的方案页 API。这种方式灵活但门槛高，适合有编程基础、想自己定制提醒规则的用户。普通用户其实没必要折腾，前面两种方式够用了。
